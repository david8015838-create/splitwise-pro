## ⚠️ 需要手動完成的項目

### GitHub Actions（可選但推薦）

由於 GitHub Token 權限限制，需要手動添加：

**文件**: `.github/workflows/keep-supabase-active.yml`

**步驟**:
1. 查看 `GitHub-Actions-手動添加說明.md`
2. 在 GitHub 網站上創建文件
3. 設置 Secrets
4. 測試執行

**預計時間**: 5 分鐘

**作用**:
- 🤖 每天自動 ping 資料庫
- 🧹 自動清理 30 天前的資料
- 🛡️ 防止 Supabase 免費版暫停
